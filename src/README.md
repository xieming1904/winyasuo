# Unpack - Windows Compression Utility

This directory will contain the source code for the Unpack application.

## Project Setup
- **Technology:** C# with WinUI 3 (Windows App SDK)
- **IDE:** Visual Studio 2022
- **Key SDKs:** Windows App SDK, .NET SDK

## Initial Structure (Conceptual)
- `Unpack.sln` (Solution File)
- `Unpack/` (Main Project Directory)
  - `Unpack.csproj` (Project File)
  - `App.xaml` / `App.xaml.cs` (Application definition)
  - `MainWindow.xaml` / `MainWindow.xaml.cs` (Main application window)
  - `Assets/` (For images, icons, etc.)
  - `Package.appxmanifest` (Application manifest)
